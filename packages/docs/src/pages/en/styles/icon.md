# Icon 图标

基本单位 P = 4px，常用图标模版为 3P、4P、6P、8P，所有线性图标的描边宽度统一为 1.5px

## 3P

当需要设计小字号的文字按钮时，使用 3P 的图标。

<icons type="3p" />

<br>

## 4P

当元素之间无法通过间距满足视觉分类时，需要在元素之间加插分隔组件，增加对比度。

<icons type="4p" />

<br>

## 6P

6P 图标常用于底部导航栏或者历史记录。

<icons type="6p" />

<br>

## 8P

8P 图标为通用图标，可以通过程序缩放到 6P，4P 尺寸，描边保持 1.5px。

<icons type="8p" />
